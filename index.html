Configuración

Conecta con Google Sheets para sincronización en la nube

🔗 Conexión vía Google Apps Script
Pasos para conectar:
1. Abre tu Google Sheet → menú Extensiones → Apps Script
2. Pega el código del archivo Code.gs que descargaste
3. Clic en Implementar → Nueva implementación
4. Tipo: Aplicación web · Ejecutar como: Yo · Acceso: Cualquier persona
5. Copia la URL de la aplicación web y pégala abajo
6. Haz clic en "Crear estructura" para generar las hojas automáticamente
📋 Estructura del Google Sheet requerida

El sistema necesita estas hojas (tabs) en tu Google Sheet:

HojaColumnasDescripción
FacturasNo_Factura, Fecha, Emisor, NomComercial, NIT, Dirección, Total, Cuenta, CUFE, ObsCabeceras de facturas
Detalle_FacturasNo_Factura, Linea, Codigo, Descripcion, Cantidad, VUnitario, BaseGravable, INC, Total, CategoriaÍtems de cada factura
IngresosFecha, Cliente, Concepto, Categoria, Cuenta, Monto, ObsRegistro de ingresos
MovimientosFecha, Tipo, Descripcion, Origen, Destino, MontoOrigen, MontoDestino, ReferenciaMovimientos entre cuentas
InsumosProveedor, Codigo, Nombre, Categoria, Subcategoria, UnidadMedidaMaestro de insumos
